# くらし道具ノート運用メモ

確認日: 2026-06-16

## サイトの位置づけ

このサイトは「自動で収益を出すサイト」として見せるのではなく、暮らしの買い物候補を整理する比較メモとして運用します。

読者向けの表現:

- 価格とレビューで比較
- 季節の買い物候補
- 販売ページで最終確認
- まとめ買い、備蓄、ギフトの候補整理

避ける表現:

- 絶対おすすめ
- 必ず得する
- 効果があると断言する健康・美容表現
- クリックや購入を不自然に促す表現

## 初期テーマ

1. 水・炭酸水・お茶のまとめ買い
2. 米・保存食・常温ストック
3. 季節ギフトと手土産候補

理由:

- レビュー件数が多く比較しやすい
- 日常需要と季節需要がある
- 医療や金融のような高リスク表現になりにくい
- 価格、容量、送料、レビューを整理する価値がある

## 季節運用

- 1月: 新年、冬の買い置き
- 2月から3月: 新生活準備
- 4月から5月: 初夏、母の日
- 6月から8月: お中元、夏ギフト、水分補給
- 9月: 防災、敬老の日
- 10月から11月: 秋冬のまとめ買い
- 12月: お歳暮、年末年始準備

## 安全運用

- 自分で広告リンクをクリックして購入しない
- 他サイトの商品説明文をコピーしない
- SNSへ大量自動投稿しない
- 広告リンクが含まれることを表示する
- 価格、在庫、送料、ポイント条件は販売ページ確認と書く

## 人間が必要な作業

- 楽天アフィリエイト、楽天Web Serviceの登録
- 本人確認、口座情報、税務情報などの入力
- GitHub SecretsへのAPIキー保存
- 実際の商品ページの表示確認

口座情報やAPIキーはチャットに貼らず、公式サイトとGitHub Secretsだけに入力します。
